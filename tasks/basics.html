HTML basics

HTML:- hyper text markup language

Basic structure of html:

Expalnation of structure of HTML:

  1. <!DOCTYPE html>
    Declares the document type and version of HTML (HTML5 here).
  2. <html lang="en">
    The root element of the HTML document. The lang attribute specifies the language of the document.
  3. <Head>
    Contains meta-information about the document (not visible on the page).
    • <meta charset="UTF-8">:
      Sets character encoding.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">:
      Ensures proper scaling on different devices.
    • <title>
      The title shown in the browser tab.
  4. <body>
    Contains the content that is visible to users:
    • <h1>
      Main heading.
    • <p>
      Paragraph of text.

Text formatting

  1. Bold:
    <b></b> create bold text. Also b can be replaced with strong
  2. Italic
    <i></i> creates italic text. Also i can be replaced with em
  3. Underline
    <u> Underlines the text where ever needed.

Creating a link:

Links can be created by using the anchor tags.

<a href="" target="">click here</a>

Lists in HTML:

  1. Ordered list
    <ol><li></li></ol> Gives list in an order
  2. Unordered list
    <ul><li></li></ul> Gives list without order
  3. Description list
    A description list is used to group terms and their descriptions. It's great for glossaries, FAQs, or metadata.
    • Baisc Tags
      • <dl>:
        Description list (container)
      • <dt>:
        Description term (like a heading)
      • <dd>:
        Description detail (explanation of the term)
